Presentazione

Ho studiato programmazione per 3 anni, prima iniziando brevemente con c++ e poi spostandomi seriamente su java, ho un'eccelente conoscenza del linguaggio e sono aggiornato a tutte le maggiori features di java 9. Conosco javascript, base e volto al functional programming, html, css, sql e le maggiori tecnologie per il web development, sia front che back end. Conosco i principali algoritmi di ordinamento, le basi software e hardware del funzionamento dei computer e ho ottime conoscenze matematiche fino ad algebra 2.
